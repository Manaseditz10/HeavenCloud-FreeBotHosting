## Common Issues & Fixes

**Bot keeps going offline?**
Most bots go offline due to hosting providers with sleep mode or insufficient RAM. Heaven Cloud allocates dedicated resources per bot so this does not happen. Read more: [Why does my Discord bot keep going offline?](https://www.heavencloud.in/blog/why-discord-bot-keeps-going-offline)

**How much RAM does a Discord bot need?**
A basic Discord bot needs around 100-256MB RAM. Bots with music, databases, or heavy commands need 512MB or more. Read more: [How much RAM does a Discord bot need?](https://www.heavencloud.in/blog/how-much-ram-discord-bot-need)

**Free vs Paid hosting — which one to pick?**
Free hosting works well for small or personal bots. For bots in 50+ servers or with high command usage, a paid plan gives more stability and resources. Read more: [Free vs Paid Discord Bot Hosting](https://www.heavencloud.in/blog/free-vs-paid-discord-bot-hosting)
